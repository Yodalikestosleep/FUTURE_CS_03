How to run?

1. Install Requirements:
    pip install -r requirements.txt

2. Create a virtual environment:
    For Linux or mac:
        python3 -m venv venv
        source venv/bin/activate
    For Windows:
        python -m venv venv
        .\venv\Scripts\activate
3. Run the file:
    python app/app.py

// The project will run on default port 5000 on localhost.
